# Nuevo_Amanecer
Sitio web oficial de Nuevo Amanecer, un emprendimiento dedicado a la creación de regalos personalizados en Puerto Montt. Desarrollado con HTML, CSS, JavaScript, Firebase y optimizado para SEO. Incluye galería dinámica, filtros por categoría, carrusel conectado a panel, contacto directo por WhatsApp y despliegue en Vercel.
